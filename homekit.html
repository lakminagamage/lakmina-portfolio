Smart Home Control Application

This project is done with Arduino CH340g ESP8266 development board. Here we've intergrated apple iOS and macOS's Home app with the wifi enable development board to control realys with mobile phone's input. And also we've intergrated this with Apple SIri to make the home equipments like Lights fans and switches voice Controlled.
